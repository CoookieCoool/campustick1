# CampusTick — Frontend

React + Vite SPA.

## Setup

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

App runs at `http://localhost:3000`

## Pages
| Route | Page |
|-------|------|
| /login | LoginPage |
| /register | RegisterPage |
| /dashboard | DashboardPage (protected) |
| /events | EventsPage |
| /events/:id | EventDetailsPage |
| /events/create | CreateEventPage (protected) |
| /tickets | TicketPage (protected) |
| /scan | ScanPage (protected) |
